\subsection{Amplitude Addressing vs Array Mutation}
Physical simulation of $N$-qubit states ordinarily requires $2^N$ complex floats, an exponentially scaling wall that renders classical representations of anything beyond 50 qubits physically impossible on Earth-bound hardware. 

The most revolutionary engineering aspect of this repository is the rejection of array representation. State vectors are never stored dynamically. Instead, complex amplitudes are constructed as associative bindings to the 96 fixed topological classes. The entirety of the state vector is represented as a single, cryptographically folded \textbf{Universal Object Reference (UOR)} hash ($\kappa$). Computations occur by feeding these $\kappa$ states into the holospaces substrate engine, which evaluates the braid permutations natively over the geometric belt without ever expanding the intermediate vector.
